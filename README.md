# Train Ticket Booking System

A Train Ticket Booking System built using:

- FastAPI
- SQLAlchemy
- SQLite
- Kafka-inspired Event Streaming
- HTML/CSS/JavaScript

## Features

- User Registration & Login
- Train Search
- Seat Availability
- Ticket Booking
- Tatkal Booking
- Web Check-in
- Boarding Slip Generation
- Kafka Producer/Consumer Simulation
- Live Analytics

## Run Backend

```bash
uvicorn main:app --reload
